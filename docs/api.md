## 4. RAG 知识库接口

### GET `/rag/collections`
获取所有知识库集合

### GET `/rag/collections/{collection_name}`
获取集合详情

### POST `/rag/documents`
添加文档到知识库

**请求体：**
```json
{
  "collection": "products",
  "text": "文档内容...",
  "metadata": {"type": "product", "sku": "12345"}
}
```

### POST `/rag/search`
语义搜索

**请求体：**
```json
{
  "collection": "products",
  "query": "油皮能用吗",
  "top_k": 3
}
```

### DELETE `/rag/collections/{collection_name}`
删除知识库集合

## 7. 错误处理与状态码

### 7.1 状态码说明

| 状态码 | 描述 | 含义 |
|--------|------|------|
| 200 | OK | 请求成功 |
| 400 | Bad Request | 请求参数错误 |
| 401 | Unauthorized | 未授权访问 |
| 404 | Not Found | 资源不存在 |
| 500 | Internal Server Error | 服务器内部错误 |
| 503 | Service Unavailable | 服务不可用（如LLM或Qdrant连接失败） |

### 7.2 错误响应格式

```json
{
  "detail": "错误描述",
  "code": "错误代码",
  "message": "友好错误信息"
}
```

### 7.3 常见错误示例

**400 Bad Request**
```json
{
  "detail": "Invalid input data",
  "code": "INVALID_INPUT",
  "message": "请求参数格式错误，请检查输入数据"
}
```

**503 Service Unavailable**
```json
{
  "detail": "LLM service is unavailable",
  "code": "LLM_UNAVAILABLE",
  "message": "大模型服务暂时不可用，请稍后重试"
}
```
